This graph shows the Confusion Matrix for the last Validation run. The X axis shows the value that the neural network predicted.
The Y axis shows the label from the training data (that is, what the actual value should be). You can see, for each
prediction, what the value should have been. For the most part, the neural network does a good job of predicting the number.
However, there are quite a few instances when it though a 4 was a 9, and vice versa.

This graph shows the Training Accuracy and the Validation Accuracy at the end of each epoch. The Training Accuracy should be similar
to the Validation Accuracy. If the Training Accuracy is significantly bigger than the Validation Accuracy, it is a sign that
the neural network is overfitting to the Training Data. If the Training Accuracy is significantly lower than the Validation
Accuracy, it is a sign that the neural network is underfitting to the Training Data.
